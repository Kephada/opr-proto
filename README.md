# One Punch Run — playtest build

Public **playtest** deploy of *One Punch Run* (working title), a one-tap
dice-as-heroes roguelike. Build-free vanilla ES-module web; static files only.

The hub menu links the base game plus four mechanic prototypes; the headline
one is **IV · Nested** (dice socket inside bigger dice and burst out as combos).

This repo holds only the playable client bundle, mirror-published by an
allowlist deploy script. Source history, design docs, and tooling live in a
separate private repository.
