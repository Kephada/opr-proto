# One Punch Run — playtest build

Public **playtest** deploy of *One Punch Run* (working title), a one-tap
dice-as-heroes roguelike. Build-free vanilla ES-module web; just static files.

- **Play:** open the GitHub Pages URL for this repo.
- The hub menu links the base game plus four mechanic prototypes; the headline
  one right now is **IV · Nested** (dice socket inside bigger dice and burst out
  as combos).

This repo holds only the playable client bundle. Source history, design docs,
and tooling live in a separate private repository.
